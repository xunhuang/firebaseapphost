# Firebase App Hosting Example

This repository contains a minimal example project configured for Firebase App Hosting based on the [Firebase App Hosting quickstart](https://firebase.google.com/docs/app-hosting/get-started).

## Project setup

1. Update `.firebaserc` with your Firebase project ID.
2. Install dependencies:
   ```bash
   npm install
   ```
3. Build the Next.js app:
   ```bash
   npm run build
   ```
4. Deploy using the Firebase CLI:
   ```bash
   firebase deploy
   ```

For more detailed instructions, refer to the official guide linked above.
